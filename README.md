# Piggy Bank – Demo Frontend

## Purpose

This is a temporary demonstration frontend for the Piggy Bank personal finance API. It serves as a functional placeholder that showcases the backend capabilities while the full-featured frontend is under development.

## What It Does

- User authentication (register/login with JWT)
- Dashboard with net worth, monthly spending, and savings rate
- Transaction management (create, view, export CSV/PDF)
- Budget tracking with progress visualization
- Savings goal monitoring
- Spending insights with category breakdown and anomaly detection
- Category management for transactions

## Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
- Bootstrap 5 for styling
- Chart.js for data visualization
- No frontend framework – keeps it lightweight and simple

## Important Note

The backend is hosted on Render's free tier. If the app appears slow or unresponsive on first load, please allow 20-40 seconds for the backend service to spin up from inactivity.

## Status

This is a placeholder frontend. The production frontend will be built with React/Vue and include additional features such as:

- Mobile-responsive design
- Offline support
- Push notifications
- Advanced reporting
- Multi-currency support
